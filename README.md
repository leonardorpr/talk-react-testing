# talk-react-testing

:squirrel: talk about react testing for the Ruptiva team

## Get started

```bash
$ yarn install
$ yarn start
```

open your browser [http://localhost:3000](http://localhost:3000) and enjoy (:

## Tests

run this command for the unit and integration tests

```base
$ yarn test
```

and for the e2e

```bash
$ yarn e2e
```
